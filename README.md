# gutenberg-blocks

Custom blocks for displaying content within Drupal.

## Our custom blocks
* Image
* CTA link
* Highlight (wrapper)
* Image gallery (overlay)
* iFrame (with 3 types)
* Checklist
* Accordion


## Deployment
* Run `npm run build` before committing
* Builed JS/CSS is loaded via custom Drupal module directly from git pages
